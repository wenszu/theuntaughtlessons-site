Send to instructor

When active, participants can send their results directly to you from their My results page. This requires an Apps Script URL from your connected Google Sheet.

How to get this URL

  1. Open your Google Sheet
  2. Click Extensions → Apps Script
  3. Paste the provided Apps Script code from the program documentation
  4. Click Deploy → New deployment
  5. Select type: Web app
  6. Set access to: Anyone
  7. Click Deploy and copy the web app URL
  8. Paste that URL into the field above
This URL acts as your private endpoint. Do not share it publicly.

Access

Members

Method A — Google sign-in (recommended for most members)

  1. Admin: Click + Add member, enter name and email, click Save
  2. Member receives: A welcome email from your Gmail with a link to the workspace
  3. Member: Clicks the link, signs in with their Google account — done
  4. No username or password needed. No Firebase email.

Method B — Username & password (non-Google accounts)

  1. Admin: Click + Add member, enter name, email, username, and password, click Save
  2. Member receives: Welcome email from your Gmail
  3. Admin: Share the username and password with the member separately (not in the welcome email)
  4. Member: Goes to the workspace, selects "Sign in with a username", enters credentials — done

Method C — Passwordless magic link (for members who cannot use Google)

  1. Admin: Click + Add member, enter name and email, check "Also send login link", click Save
  2. Member receives two emails:
    • Welcome email: Your welcome email from Gmail
    • Magic link: A one-click login link from Firebase (noreply@the-untaught-lessons.firebaseapp.com)
  3. Member: Clicks the Firebase link — lands directly in the workspace
  4. The Firebase link expires after 24 hours. If it expires, use "Send login link" in the table row.

Removing a member

  1. Admin: Find the member in the table, click Remove
  2. Automated: Removed from Firestore member list and user profile — their next page load will be rejected
  3. Automated after Apps Script setup: Removed from the Google Group at utl-members
  4. Active sessions last up to 1 hour before expiring. Outstanding login links cannot be remotely revoked.

GitHub setup

Connect to GitHub so this admin panel can read and write data files directly. Your token is stored only in this browser — it never goes into the code or git history.

Step 1 — Create a fine-grained token.
Go to github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token.
Set repository access to Only select repositories → theuntaughtlessons-site.
Under Permissions → Repository permissions, set Contents → Read and write. Everything else stays None.

Step 2 — Paste the token below and click Save.
