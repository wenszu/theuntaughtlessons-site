How to transcribe the voice file

How to transcribe the voice file

  1. Open the voice file: You were sent a voice memo file via email. Open it on your phone or laptop.
  2. Open ChatGPT voice mode: Go to chatgpt.com. Click the microphone icon to start voice input.
  3. Play the file into ChatGPT: Play the voice memo out loud while ChatGPT listens through your microphone. It transcribes in real time.
  4. Copy the transcript: Once the file ends, stop ChatGPT. Select all the transcribed text and copy it.
  5. Paste and use the AI prompt: Paste the transcript into ChatGPT. Then use the AI structuring prompt below to have it structure the memo into MECE buckets.

Alternative free tools if ChatGPT is unavailable:

  • Google Docs voice typing: Use Tools → Voice typing. Free, works in Chrome.
  • Otter.ai free tier: Upload your audio file directly. It transcribes automatically.
  • Manual fallback: If transcription fails, type what you heard. Minimum 50 words to continue.
