# Visual system (from slides)

## Core feel
- Calm, minimal, structured
- Feels like a thinking tool, not a marketing page
- Every slide communicates ONE idea only

## Layout patterns

### Titles
- Large serif headline (Playfair)
- Center or top-left aligned
- Very wide spacing around text

### Sections
- Clean divider slides with almost no content
- Used to reset attention

### Content slides
- Short paragraphs or structured bullets
- No dense blocks
- Left aligned

### Framework slides
- Simple diagrams (pyramid, trees, 3 pillars)
- Hand-drawn or whiteboard style
- Visual supports thinking, not decoration

### Exercise slides
- Clear instructions
- Structured lists (MECE, BSP)
- Feels interactive, like a worksheet

## Spacing rules
- Heavy whitespace (40%+ empty)
- Strong margins
- No clutter

## Color usage
- Cream background (default)
- Navy for text
- Gold used VERY sparingly (highlight only)

## Typography
- Serif (Playfair) for headlines
- Sans (Lato) for body
- Monospace for AI prompts

## What to avoid
- Dense UI sections
- Multiple competing elements
- Card-heavy layouts
- SaaS-style grids
